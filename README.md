# 📊 E-commerce Order Lifecycle & Performance Analysis

## 📌 Overview

This project analyzes an e-commerce dataset of **89K+ orders** to understand order lifecycle behavior, cancellation patterns, and overall operational performance. The goal is to identify trends in order volume, fulfillment efficiency, and factors influencing cancellations.

---

## 🎯 Objectives

* Analyze order lifecycle stages from purchase to delivery
* Evaluate cancellation patterns and monthly trends
* Measure key business metrics such as order volume and cancellation rate
* Identify operational insights related to order processing and fulfillment

---

## 🛠️ Tools & Technologies

* **Excel** – Data cleaning and preprocessing
* **MySQL** – Data analysis and query execution
* **Power BI** – Dashboard creation and visualization

---

## 📂 Dataset

* Dataset contains **~89,000 e-commerce orders**
* Includes fields such as:

  * Order ID
  * Customer ID
  * Order Status
  * Purchase Timestamp
  * Approval Timestamp
  * Delivery Timestamp
  * Approval Delay
  * Cancellation Indicator

📎 Dataset Source:
Olist Brazilian E-commerce Dataset (Kaggle)

---

## 🔍 Key Analysis Performed

* Order status distribution analysis
* Monthly order trend analysis
* Monthly cancellation analysis
* Cancellation rate calculation
* Approval delay impact exploration

---

## 📈 Key Insights

* Total orders analyzed: **89K+**
* Overall cancellation rate: **~0.46%**
* Approximately **98% of orders were successfully delivered**
* Order volume showed significant growth from **2016 to 2018**
* High cancellation percentages in early months were due to **low order volume (data sparsity)**, not operational issues
* Final month dip identified as **incomplete data**, not actual decline
